About My Project

Title: Depth-Enhanced Change Detection: Evaluating the Contribution of 3D Information in Remote Sensing Imagery Analysis

Objective

The objective of this project is to detect and visualize changes between satellite images taken at different time intervals. By using advanced image processing and integration of 3d depth maps, we identify and highlight differences in terrain, infrastructure, or vegetation that may have occurred over time.

Technologies Used

  • Python for backend logic and image processing
  • OpenCV for depth map generation and image analysis
  • Matplotlib and Plotly for visualization
  • HTML, CSS for UI Design

Working Mechanism

  1. Depth Map Generation: Two satellite images are analyzed to create depth maps, which measure the relative distance of objects from the camera/satellite perspective.
  2. Change Detection: Differences between the two depth maps are computed to identify significant changes in the environment (e.g., construction, deforestation).
  3. Visualization: The results are visualized via interactive maps or 3D plots to make interpretation easier for decision-makers.

Applications

  • Urban development tracking
  • Disaster impact assessment (e.g., floods, earthquakes)
  • Agricultural monitoring and forest management
  • Border surveillance and national security

Conclusion

This system provides a reliable and scalable method for identifying land changes using computer vision and 3D modeling. It is useful for government agencies, researchers, and organizations that rely on satellite surveillance for planning, protection, and prediction.
